# Data Cleaning & Reporting Automation Using Python and Power BI

## Project Overview

This project automates the process of cleaning employee data and generating interactive reports. Python is used for data preprocessing tasks such as handling missing values and removing duplicate records, while Power BI is used to create dashboards and visual reports.

## Objective

* Clean and preprocess raw employee data.
* Handle missing values and duplicate records.
* Automate data preparation using Python.
* Generate visual reports using Power BI.
* Improve reporting efficiency through automation.

## Tools Used

* Python (Pandas)
* Microsoft Excel
* Power BI Desktop

## Dataset

The dataset contains employee information including:

* EmployeeID
* Name
* Department
* Salary

The dataset includes intentionally introduced missing values and duplicate records to demonstrate data cleaning techniques.

## Data Cleaning Process

### 1. Remove Duplicate Records

Duplicate employee records are identified and removed using Python.

### 2. Handle Missing Values

* Missing names are replaced with "Unknown".
* Missing salaries are replaced with the average salary value.

### 3. Data Validation

The cleaned dataset is verified to ensure consistency and correctness.

## Automation Workflow

Raw Employee Data (CSV)
↓

↓

↓

Automated Reports and Visualizations
